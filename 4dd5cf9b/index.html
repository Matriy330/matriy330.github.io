void mpz_powm (mpz_t rop, const mpz_t base, const mpz_t exp, const mpz_t mod) [Function]
Set rop to base^exp mod mod.
其实就是计算 base 的 exp 次方，并对 mod 取模，最后将结果写入 rop 中

rop = (base ^ exp) % mod

v7 = (v6^v5^) % v4，求v6

Vip破解

我们不应该hook isVip()，为什么在另一篇里讲过了

ps:frida -U -p 13937 -l xiaoxiguan.js

要用attach去hook而非spwn

because:

  1. Frida 启动 App 的进程
  2. App 还没开始加载 dex
  3. Frida 就执行你的脚本
  4. 此时 Java 层 尚未初始化
  5. 所以 UserInfo、com.itally.* 全部都还没加载
  6. 因此 Java.use("xxxx") → ClassNotFound

APP还没加载 Java 类时去 Hook，当然找不到。

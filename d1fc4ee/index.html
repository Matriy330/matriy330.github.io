йҖҶеҗ‘ж”»йҳІдё–з•ҢCTFзі»еҲ—7-logmein

==йҮҚиҰҒзҹҘиҜҶзӮ№пјҡдёҚеҗҢidaзүҲжң¬зҡ„еҸҚзј–иҜ‘з»“жһңдёҚеҗҢ!!!==

64дҪҚпјҢmainеҮҪж•°

зӣ®зҡ„жҳҜжү§иЎҢжңҖеҗҺзҡ„иҝҷж®өд»Јз Ғ

е®Ўи®Ўд»Јз ҒпјҢv3й•ҝеәҰ>=v8й•ҝеәҰ

if ( i >= strlen(v8) ) sub_4007C0(); еҲҷеҸҜзҹҘv3й•ҝеәҰеҝ…йЎ»зӯүдәҺv8й•ҝеәҰпјҢеҗҰеҲҷдёӢж Үiдјҡ>=v8

==зңӢеҲ°дёӢйқўйӮЈж®өд»Јз Ғзҡ„ж—¶еҖҷеҸ‘зҺ°жҖҺд№ҲйғҪи§ЈйҮҠдёҚйҖҡпјҢеҗҺжқҘеҸ‘зҺ°дәҶдёҖдёӘйңҮжғҠе°ҸзҷҪзҡ„дәӢпјҢе°ұжҳҜдёҚеҗҢзүҲжң¬зҡ„idaеҸҚзј–иҜ‘зҡ„дјӘд»Јз ҒдёҚеҗҢпјҢжҲ‘еҲҡжүҚз”Ёзҡ„жҳҜidapro 8.3 з„¶еҗҺеҺ»иҜ•дәҶidapro7.7 е’Ңidapro 7.0пјҢеҸ‘зҺ°7зҡ„еҸҚзј–иҜ‘з»“жһңжүҚжҳҜжӯЈзЎ®зҡ„пјҢеҰӮдёӢеӣҫ==

з”ЁдёҠдәҶv6е’Ңv7пјҢиҖҢд№ӢеүҚзҡ„дјӘд»Јз ҒйғҪжІЎжңүз”ЁдёҠ!!!!

и®©жҲ‘д»¬еҶҚи§ЈиҜ»дёҖдёӢдёӢйқўйӮЈдёӘиҜӯеҸҘеҗ§пјҡ

йҮҚзӮ№жҳҜпјҡ(_BYTE *)&v7зҡ„ж„ҸжҖқжҳҜпјҢжҠҠlonglongеһӢзҡ„v7ејәеҲ¶иҪ¬еҢ–дёәbyteеһӢзҡ„ең°еқҖпјҢз®ҖеҚ•зҡ„иҜҙпјҢе°ұжҳҜжҠҠе®ғзңӢжҲҗеӯ—з¬ҰдёІпјҲCиҜӯиЁҖеӯ—з¬ҰдёІжң¬иҙЁйғҪжҳҜжҢҮй’ҲйҰ–ең°еқҖ+еҒҸз§»пјүгҖӮ

з”Ёv7зҡ„еҖј10иҝӣеҲ¶иҪ¬16иҝӣеҲ¶пјҢеҫ—еҲ°пјҡebmarah

иҝҷдёӘйўҳзӣ®йҷӨдәҶеҹәжң¬зҡ„IDA proзҡ„дёҖдәӣе°Ҹе°Ҹзҡ„еҝ«жҚ·й”®д№ӢеӨ–пјҢиҝҳйңҖиҰҒзҹҘйҒ“е°Ҹз«ҜеәҸе’ҢеӨ§з«ҜеәҸзҡ„зҹҘиҜҶпјҢжҖҺд№ҲеҺ»еҲӨж–ӯжҳҜе°Ҹз«ҜеәҸе’ҢеӨ§з«ҜеәҸпјҢеӨ§жҰӮе°ұжҳҜеңЁеҒҡйўҳзҡ„ж—¶еҖҷпјҢеҰӮжһңзў°еҲ°жҲ‘д»¬йңҖиҰҒжүӢеҠЁеҺ»иҪ¬еӯ—з¬Ұзҡ„пјҢеә”иҜҘе°ұжҳҜеҸҚиҪ¬дёҖдёӢеӯ—з¬Ұзҡ„дҪҚзҪ®пјҢе®һеңЁдёҚиЎҢеӨ§з«ҜеәҸе’Ңе°Ҹз«ҜеәҸйғҪеҺ»иҜ•дёҖдёӢгҖӮ

еҸҚиҪ¬д»–пјҡharambe

и§ЈйҮҠдёӢиҝҷж®өи„ҡжң¬пјҢдёӨдёӘеӯ—з¬ҰдёҚиғҪзӣҙжҺҘејӮжҲ–еӣ жӯӨиҰҒз”ЁordпјҢе°Ҷеӯ—з¬ҰиҪ¬дёәеӯ—з¬ҰеҶҚејӮжҲ–пјҢеҶҚchrиҪ¬еҲ°еӯ—з¬Ұ

еҫ—еҲ°RC3-2016-XORISGUD

chr() з”ЁдёҖдёӘиҢғеӣҙеңЁrange(256)еҶ…зҡ„пјҲе°ұжҳҜ0~255пјүж•ҙж•°дҪңеҸӮж•°пјҢиҝ”еӣһдёҖдёӘеҜ№еә”зҡ„еӯ—з¬ҰгҖӮ
еҸӮж•°пјҡiвҖ”еҸҜд»ҘжҳҜ10иҝӣеҲ¶д№ҹеҸҜд»ҘжҳҜ16иҝӣеҲ¶зҡ„еҪўејҸзҡ„ж•°еӯ—
иҝ”еӣһеҖјпјҡеҪ“еүҚж•ҙж•°еҜ№еә”зҡ„ASCIIеӯ—з¬ҰпјҢеҲҶеҲ«иҫ“е…ҘеҚҒиҝӣеҲ¶ж•°е’ҢеҚҒе…ӯиҝӣеҲ¶ж•°

ord()еҮҪж•°жҳҜchr()еҮҪж•°зҡ„й…ҚеҜ№еҮҪж•°пјҢе®ғд»ҘдёҖдёӘеӯ—з¬ҰдҪңдёәеҸӮж•°пјҢиҝ”еӣһеҜ№еә”зҡ„ASCIIж•°еҖјгҖӮ
иҝ”еӣһеҖјпјҡеҜ№еә”зҡ„еҚҒиҝӣеҲ¶ж•ҙж•°

hex()еҮҪж•°з”ЁдәҺе°Ҷ10иҝӣеҲ¶ж•ҙж•°иҪ¬жҚўдёә16иҝӣеҲ¶пјҢд»Ҙеӯ—з¬ҰдёІеҪўејҸиЎЁзӨәгҖӮ

oct() еҮҪж•°е°ҶдёҖдёӘж•ҙж•°иҪ¬жҚўжҲҗ8иҝӣеҲ¶еӯ—з¬ҰдёІгҖӮ
